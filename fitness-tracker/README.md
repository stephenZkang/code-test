# Fitness Tracker - 运动健康助手

一个基于React开发的现代化运动健康管理Web应用，采用玻璃质感UI设计，帮助用户管理体重、记录跑步，并提供个性化健康方案推荐。

## ✨ 功能特点

### 🏃‍♂️ 核心功能
- **体重管理**：记录体重变化，计算BMI，追踪体重趋势
- **跑步记录**：记录跑步距离、时间、配速，内置秒表功能
- **健康方案**：基于用户数据提供个性化健康建议

### 🎨 设计特色
- **玻璃质感UI**：现代化的毛玻璃效果设计
- **响应式布局**：适配桌面和移动设备
- **简洁操作**：直观的用户界面，易于使用
- **数据可视化**：简洁直观的图表展示

### 💾 数据管理
- **本地存储**：数据自动保存到浏览器本地
- **实时统计**：自动计算各项健康指标
- **历史记录**：完整的历史数据追踪

## 🚀 快速开始

### 环境要求
- 现代浏览器（支持ES6+）
- 无需安装，直接在浏览器中运行

### 运行方法
1. 克隆或下载项目
2. 在项目目录运行：
   ```bash
   npm run dev
   ```
3. 在浏览器中访问 `http://localhost:3000`

或者直接打开 `index.html` 文件。

## 📱 使用指南

### 体重管理
1. 输入当前体重和身高
2. 系统自动计算BMI指数
3. 查看体重变化趋势图
4. 设定目标体重

### 跑步记录
1. 使用内置秒表计时跑步
2. 输入跑步距离和时间
3. 系统自动计算配速
4. 查看跑步历史和统计数据

### 健康方案
1. 选择健康目标（减脂/增肌/耐力/综合）
2. 获取个性化饮食和运动建议
3. 查看基于用户数据的特别提醒

## 🛠️ 技术栈

- **前端框架**：React 18
- **开发语言**：JavaScript (ES6+)
- **样式设计**：CSS3 + 玻璃质感UI
- **数据存储**：浏览器 LocalStorage
- **开发工具**：Babel (浏览器端编译)

## 🎯 功能特色详解

### 玻璃质感设计
- 使用 `backdrop-filter` 实现毛玻璃效果
- 半透明背景配合模糊效果
- 优雅的动画和过渡效果

### 数据可视化
- 简洁的柱状图展示趋势
- 响应式图表设计
- 实时数据更新

### 智能推荐系统
- 基于用户运动数据
- 个性化健康建议
- 多目标支持（减脂/增肌/耐力）
